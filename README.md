# Python-basic
This repository contains two Python scripts created for Assignment 1 (Module 2: Basic Python Concepts).

## Task 1: Basic Mathematical Operations
- Takes two numbers as input.
- Performs addition, subtraction, multiplication, and division.
- Displays the results of each operation.

## Task 2: Personalized Greeting
- Takes first name and last name as input.
- Combines them to form a full name.
- Prints a personalized greeting message.

Both scripts are included in this repository:
- Task1.py
- Task2.py
